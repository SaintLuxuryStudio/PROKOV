# Дело Пракова

Одностраничное интерактивное досье в формате расследования.

## Запуск

```bash
npm install
npm run dev
```

Сайт будет доступен на `http://localhost:3000`.

## Проверки

```bash
npm run test
```

Тесты проверяют:
- отсутствие запрещенных кино-маркеров в пользовательских текстах;
- целостность связей между инцидентами, уликами и таймлайном;
- наличие раскрытия всех кодов `[ДОК/ПРОТ]` в архивных сносках.
